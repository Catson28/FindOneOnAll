### Flexbox
1. **Container Flexível**:
   ```css
   .container {
       display: flex;
       flex-wrap: wrap;
       justify-content: center;
   }
   ```
2. **Item Flexível**:
   ```css
   .item {
       flex: 1;
       margin: 10px;
   }
   ```
